      ## My Programming Journey
      
      I first got access to a computer when I was a little kid, and Scratch was my introduction to programming.

      At the end of 2020, I started learning programming with Python and Java because I wanted to make Minecraft mods. Back then, I didn't even understand what OOP was. Writing Java code and getting a Gradle project to run felt incredibly difficult.

      Eventually, I gave up on Java for a while and started writing some terrible Python scripts to scrape things from the internet. Those small experiments helped me understand what programming was really about.

      In 2021, I still wanted to make Minecraft mods, so I started using MCreator. To be honest, I don't think it's a very good tool. I spent more than half a year building questionable mods and unfinished projects before finally abandoning it and returning to Python.

      By late 2021 and throughout 2022, I started using NoneBot and deployed a zhenxun_bot instance. Influenced by the NoneBot ecosystem, I also used NonePrompt to build a tool that helped MCreator users set up their mod development environments, especially the infamous `gradlew setupDecompWorkspace` process.

      *(NoneBot's codebase is excellent. Its design and coding style still influence how I write Python today.)*
